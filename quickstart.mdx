## Installation

<CodeGroup>

```bash npm
npm install limejourney
```

```bash yarn
yarn add limejourney
```

```bash pnpm
pnpm add @limejourney/sdk
```

</CodeGroup>

## Initialize the SDK

```typescript
import { LimeJourney } from '@limejourney/sdk';

const client = new LimeJourney({
  apiKey: 'your-api-key'
});
```

## Track Your First Event

```typescript
await client.events.create({
  entityId: 'user-123',
  event: 'sign_in',
  properties: {
    device: 'mobile',
    location: 'US'
  }
});
```
